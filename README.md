### install pure prompt

```
npm install --global pure-prompt
```
<https://github.com/sindresorhus/pure#npm>


put following at the end of `.zshrc`

```
autoload -U promptinit; promptinit
prompt pure
```
<https://github.com/sindresorhus/pure#getting-started>
